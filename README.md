## Sintaxis:

* Fijar_salida_digital(salida, valor)
    * salida: 1-4
    * valor: 0-1 / True-False

* Fijar_salida_analogica(salida, valor)
    * salida: 1-4
    * valor: 0-1023

* Leer_entrada_digital(entrada)
    * entrada: 1-4
    * retorno: 0-1

* Leer_entrada_digital(entrada)
    * entrada: 1-4
